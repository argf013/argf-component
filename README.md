
# argf-react

**argf-react** is a collection of reusable React components, built with TypeScript and Vite.

## Available Components

1. **Button**
2. **Dialog**
3. **DialogConfirm**
4. **Toast**
## Installation

To install the dependencies, run the following command:

```sh
pnpm install
```
## Development

To start the development server, run:
```
pnpm  dev
```
