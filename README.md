# argf-react

**argf-react** is a collection of reusable React components, built with
TypeScript and Vite.

## Available Components

1. **Accordion**
2. **Button**
3. **Dialog**
4. **DialogConfirm**
5. **Dropdown**
6. **Menu**
7. **Toast**
8. **Input**

## Installation

To install, run the following command:

```sh
pnpm install argf-react
```
